## What this is

This project uses Distributed Acoustic Sensing (DAS) on the IRIS submarine fiber optic cable at Þorlákshöfn, Iceland, to:

1. **Classify** the current sea state as calm, swell, or stormy (92% accuracy)
2. **Forecast** significant wave height Hₛ up to 24 hours ahead (R² > 0.70 at all horizons)

No wave buoy is required. The only inputs are the cable signal and wind observations from a nearby station.

## How to run

### Environment setup
```bash
conda activate das
cd /mnt/c/Users/Hanne/Documents/verkx
```

### Step 1 — Extract DAS features
Reads all raw pkl files and computes 7 base features per 5-minute window, saved to `das_features.csv`.
```bash
python das_wave_pipeline.py
```

### Step 2 — Classification
Trains a Random Forest classifier using k-means class boundaries and Extra Trees feature selection. Outputs classification results and figures.
```bash
python classification_final_v3.py
```

### Step 3 — Forecasting
Trains SVR (h ≤ 3h) and Gradient Boosting (h > 3h) models for all forecast horizons. Outputs R²/RMSE results and timeseries figures.
```bash
python forecasting_final_v2.py
```

### Step 4 — Single launch forecast
Runs a single forecast launch with 10–90% bootstrap confidence interval (n=50). Takes ~15 minutes.
```bash
python single_launch_forecast_v2.py
```

| Split | Days | Records |
|-------|------|---------|
